# A learning project on writing your own bundles for ADCM

## Ресурсы для изучения и работы

- [Arenadata — ведущий разработчик платформы сбора и хранения данных](https://arenadata.tech/)
- [Универсальный оркестратор гибридного ландшафта](https://arenadata.tech/products/arenadata-cluster-manager/)
- [Arenadata Cluster Manager](https://docs.arenadata.io/ru/ADCM/current/introduction/intro.html)
- [Ansible Best Practices: The Essentials](https://www.ansible.com/blog/ansible-best-practices-essentials)
- [Ansible: Up and Running, 3rd Edition](https://www.oreilly.com/library/view/ansible-up-and/9781098109141/)

## Разработка, Путевые заметки

За основу моего бандла возьмём код из документации [Создание простого бандла](https://docs.arenadata.io/ru/ADCM/current/how-to/simple-bundle.html#%D0%BF%D1%80%D0%B8%D0%BC%D0%B5%D1%80-%D1%84%D0%B0%D0%B9%D0%BB%D0%B0-config-yaml) поэтапно разберём каждый объект.
